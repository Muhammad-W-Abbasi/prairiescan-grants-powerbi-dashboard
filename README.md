# PrairiesCan – Grants & Contributions Dashboard

## Overview

This project analyzes publicly available PrairiesCan Grants & Contributions data using Power BI. The objective was to design an executive-level dashboard that provides insight into funding distribution, trends, and regional allocation.

## Dataset

Source: PrairiesCan public Grants & Contributions dataset (CSV)
Scope: 2015–2025 agreements
Records: ~3,000+

## Data Preparation

- Cleaned and filtered relevant columns in Power Query
- Standardized data types (currency, dates, text)
- Created Date column for time-based analysis
- Built DAX measures for aggregation and year-over-year comparison

## Key Measures

- Total Funding
- Project Count
- Average Agreement Value
- Funding (Previous Year)

## Dashboard Features

- Executive KPI summary
- Funding trend over time
- Geographic distribution by province
- Ranked funding by province

## Tools Used

- Power BI Desktop
- Power Query
- DAX
